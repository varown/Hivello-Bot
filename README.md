## 🌟 支持

- 支持代理，配置在`proxy.txt`
- 支持多账号,配置在`devices.txt`


## 📋 运行条件

- Node.js (v16 or higher)
- npm (Node Package Manager)


## 🛠️ 初始化

1. 下载代码:
```bash
git clone https://github.com/varown/Hivello-Bot.git
cd Hivello-Bot
```

1. 安装依赖:
```bash
npm install
```


## 运行
```bash
npm run start
```
